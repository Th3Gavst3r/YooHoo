# YooHoo

A Discord bot which saves posted YouTube videos to playlists.

## Commands

| **Command**                           | **Description**                              |
| ------------------------------------- | -------------------------------------------- |
| **`yh help`**                         | Get help with bot commands                   |
| **`yh register (all) <playlist id>`** | Register a playlist with the current channel |
